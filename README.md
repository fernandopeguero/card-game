# Card Game 

This is going to be a card game using a online cdn provided for the card and displaying then using react.
the game won't be to complex is a simple memory game.

## Rules 

- Memorize the card that you have click on.
- Do not click on the same card twice. 
- Beat your previous high score

I want to make the game fun. I'm going to implement audiofx for the card flips and maybe some nice movements for the card and other things let's see how it goes.
